原理

registry

在了解了 create_model 函数的基本使用之后，我们来深入探索一下 create_model 函数的源码，看一下究竟是怎样实现从模型到特征提取器的转换的

https://blog.csdn.net/weixin_44966641/article/details/121364784

在 timm 内部，有一个字典称为 _model_entrypoints 包含了所有的模型名称和他们各自的函数。比如说，我们可以通过 model_entrypoint 函数从 _model_entrypoints 内部得到 xception71 模型的构造函数。

如我们所见，在 timm.models.xception_aligned 模块中有一个函数称为 xception71 。类似的，timm 中的每一个模型都有着一个这样的构造函数。事实上，内部的 _model_entrypoints 字典大概长这个样子：

view code 不适合放代码，都锁在一起了

在 timm 对应的模块中，每个模型都有一个构造器。比如说 ResNets 系列模型被定义在 timm.models.resnet 模块中。因此，实际上我们有两种方式来创建一个 resnet34 模型

import timm from timm.models.resnet import resnet34 # 使用 create_model m = timm.create_model(‘resnet34’) # 直接调用构造函数 m = resnet34()

但使用上，我们无须调用构造函数。所用模型都可以通过 create_model 函数来将创建。

下载pretrained

pretrained

如果我们传入 pretrained=True，那么 timm 会从对应的 URL 下载模型权重参数并载入模型，只有当第一次（即本地还没有对应模型参数时）会去下载，之后会直接从本地加载模型权重参数。

True:timm直接根据对应的URL下载模型权重参数并加载到模型，注意，只有本地没有对应模型参数时才会下载，也就是说，通常在第一次运行时下载对应模型参数，之后会直接从本地加载模型权重参数。（报错经验：这里容易出现的报错，就是第一次下载时，可能因为网络等原因，下载不完全，因此，有时需要删除对应文件，重新下载）

新手友好的深度学习CV入门教程-timm篇

本文介绍一下简单的使用

简单常用用途

https://timm.fast.ai/training
基础介绍，了解有关 timm 内部结构的更多信息

查看模型信息

创建模型

创建模型的最简单方法是使用create_model; 一个可用于在 timm 库中创建任何模型的工厂函数（factory function）。这个函数各个参数有什么用，具体怎么实现的，怎么玩出花来先不用管，挖坑进阶提高

model_resnet34 = timm.create_model(‘resnet34’, pretrained=True)
model_resnet34#打印出模型结构
model_resnet34.default_cfg#访问它的配置，其中包含诸如应该用于规范化输入数据的统计信息、输出类的数量和网络分类部分的名称等信息.

timm 模型有方法get_classifier ，我们可以使用它来检索分类头，而无需查找模块名称

微调模型-自己运行跑一跑resnet34

我的模型，用不了，维度不匹配
比如说我在resnet34模型传入cifar10和imagenet，分类类别不一样，输入的图片大小不一样
0.正常运行
torch.randn:用来生成随机数字的tensor，这些随机数字满足标准正态分布（0~1）
使用随机张量表示单通道图像，我们可以看到模型已经处理了图像并返回了预期的输出形状。

import torch
x = torch.randn([1, 3, 224, 224])
model_resnet50 = timm.create_model(‘resnet50’, pretrained=True)
out = model_resnet50(x)
print(out.shape)# Results: torch.Size([1, 1000])

1.输入维度：改变输入通道数：in_chans
挖坑引申进阶提高https://timm.fast.ai/models#So-how-is-timm-able-to-load-these-weights?

这些来自torchvision的预训练权重不能用于单通道输入图像。作为解决方案，大多数从业者通过复制单通道像素来创建3通道图像，从而将其单通道输入图像转换为3通道图像。

timm has a way to handle these exceptions

通过将in_chans_参数传递给 来指定输入图像的通道数
通道数改变后，对应的权重参数会进行相应的处理，此处不作详细说明，
可参照：https://fastai.github.io/timmdocs/models或直接查看源代码
x = torch.randn([1, 1, 224, 224])
feature_extractor = timm.create_model(‘resnet50’,pretrained=True, in_chans=1)

2.改变输出类别数目，微调模型：num_classes

如果设置num_classes，表示重设全连接层，该操作通常用于迁移学习

model_resnet50_finetune = timm.create_model(‘resnet50’, pretrained=True, num_classes=10)
out = model_resnet50_finetune (x)
print(out.shape)# Results: torch.Size([1, 10])
